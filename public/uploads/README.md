# Drop your photos here

This is the **only** folder you need. Drop any number of photos in here and the
landing page will use them automatically.

## How (no command line)

1. Open this folder on GitHub:
   **https://github.com/royaleagleweb/bathroomremodel/upload/main/public/uploads**
2. Drag your photos in from Finder / File Explorer.
3. Scroll down, leave the commit message alone, click **Commit changes**.
4. Cloudflare auto-rebuilds in 1–2 minutes. Refresh the site.

## How the photos are used

Photos are picked in **alphabetical order** of filename. So if you want a
specific photo as your hero, name it something that sorts first
(e.g. `01-hero.jpg`).

- 1st photo → hero background (top of page) and CTA banner background
- 2nd–4th photos → service cards
- 5th–10th photos → portfolio teaser grid
- If you upload fewer photos, they repeat to fill the slots.

## What works

- JPG, PNG, WebP, AVIF, GIF
- Any filename, any size — but keep each file under **2 MB** for page-speed.
  Compress at https://squoosh.app if needed.
